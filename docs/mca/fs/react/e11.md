# Question 1

Develop a React-JS application using a functional component that 
* Renders a list of countries with their capitals using `.map()`.
* Each list item must have a unique key
* List should be styled using an external CSS file.

## `CountryList.jsx`

```jsx
import React from "react";
import "./country.css";

// Constant data outside the component
const countries = [
  { id: "IN", name: "India", capital: "New Delhi" },
  { id: "US", name: "United States", capital: "Washington, D.C." },
  { id: "CA", name: "Canada", capital: "Ottawa" },
  {id: "GE", name: "Germany", capital: "Berlin" },
];

function CountryList() {
  return (
    <div className="country-list-container">
      <h2>Countries and Their Capitals</h2>
      <ul className="country-list">
        { countries.map( ({ id, name, capital }) => (
          // Destructuring each object
          <li key={id} className="country-item">
            <strong>{name}</strong>: {capital}
          </li>
        ))}
      </ul>
    </div>
  );
}
export default CountryList;
```

## Tricks in map

Both of these can work even if nested and having a nesting return.

`() => (...)` Implicitly return JSX 
```jsx
{countries.map(({ id, name, capital }) => (
  <li key={id}>
    {name} : {capital}
  </li>
))}
```

`() => {...}` needs Explicit return since it is a function body
```jsx
{countries.map(({ id, name, capital }) => {
  return (
    <li key={id}>
      {name} : {capital}
    </li>
  );
})}
```
